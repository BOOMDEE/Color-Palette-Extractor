# **App Name**: Color Palette Extractor

## Core Features:

- Image Selection: Image upload/selection: Allow the user to upload an image or select one from their device.
- Color Extraction: Color extraction: AI tool to extract the main colors from the image. Limit to 5-8 prominent colors for a balanced palette.
- Palette Display: Palette display: Display the extracted colors in a visually appealing format, showing the color swatches and their corresponding HEX codes.
- Color Code Copy: Color code copying: Allow users to easily copy the HEX codes of the extracted colors.
- Palette Saving: Color palette saving: User to locally save color palette. This saved color palette will be displayed as the user saves it in the system.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to reflect a sense of calm and subtlety in the extraction process.
- Background color: Very light gray (#F5F5F5), providing a neutral backdrop to emphasize the extracted colors.
- Accent color: Muted violet (#9400D3). This color will stand out from the other UI elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and clean interface.
